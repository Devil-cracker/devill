# 🛡️ DEVIL Scanner UI

A modern **Cybersecurity Scanner Dashboard UI** built using HTML, CSS, and JavaScript.
Designed to simulate real-world security testing tools with a clean terminal-style interface and interactive dashboard.

---

## 🚀 Features

* 🔍 Recon, SAST, and DAST simulation
* 💻 Terminal-style live output display
* 📊 CVSS-based vulnerability scoring system
* ⚡ Real-time animated dashboard UI
* 🎯 Domain validation & basic security checks
* 🎨 Advanced UI with smooth animations

---

## 🧰 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## ▶️ Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/devil-scanner-ui.git
   ```

2. Navigate to the project folder:

   ```bash
   cd devil-scanner-ui
   ```

3. Open the file:

   ```
   index.html
   ```
